## Conway's Game of Life

Written in TypeScript, React, Tailwind, and [`regl`](https://www.npmjs.com/package/regl)

Supports custom rules and [RLE](https://conwaylife.com/wiki/Run_Length_Encoded) objects

[Wikipedia article](https://en.wikipedia.org/wiki/Conway%27s_Game_of_Life)

### Running in the browser

- [GitHub Pages deployment](https://idkidk000.github.io/game-of-life/)

### Running the dev server locally

- Clone the repo

- Install dependencies with `npm install`, `deno install`, etc

- Run the `icons` script to generate icons with `npm run icons`, `deno task icons`, etc

- Run the `dev` script to start the dev server with `npm run dev`, `deno task dev`, etc
